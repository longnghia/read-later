# Read Later

------------
Read Later is an extension to manage your bunch of browser's tabs.
It keeps them in a vault that you can recall later

## Features

### Save tabs

- Current tab with `ctrl + b`
- Hightlighted tabs

### Save groups

- Save group of related tabs

### Export JSON

- Export you DATA

### Open your json links
